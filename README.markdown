W3C Validator Suite
---------------

### Play 2.1-SNAPSHOT

```
git clone git://github.com/playframework/Play20.git
cd Play20/framework
./build publish-local
```

### start geeking

You need Java 7. First build will take time as it fetches all the dependencies.

```
git clone git@github.com:w3c/validator-suite.git
cd validator-suite
<path-to-Play20>/play
```
