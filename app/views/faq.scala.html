What is the W3C Validator Suite®?

The W3C Validator Suite® service checks an entire Web site and evaluates conformance with W3C open standards. Validation currently includes HTML5, CSS and Internationalization checking.

How can I use Validator Suite service?

W3C is providing Validator Suite as Software-as-a-Service. It is offered completely online as a One-Time service.

What is the difference between the existing free W3C Markup Validator and this new service?

Validator Suite integrates Web standards conformance tools to make it easy and quick to keep your entire Web site compliant with the latest Web standards.

W3C will continue to offer its free validator tools, including the single page Markup Validator. This free service only checks one Web page at a time.

When I see the W3C Validator Suite® logo/badge, what does it mean?

The service logo refers to the five key benchmarks for the Open Web Platform. The Validator Suite badge says "Improved with W3C Validator Suite®". You are encouraged to post the badge to your Websites to indicate you used the service, although it does not imply any seal of approval or service outcomes.

Do W3C Members enjoy special privileges?

Yes, a benefit of W3C Membership is that Members do receive a Validator Suite service discount. Please contact our sales team directly instead of using the online payment platform.

What about Accessibility testing?

Currently the W3C Validator Suite® does not test for accessibility. Additional conformance checking for accessibility, mobile, etc. is under consideration for the near future.

How long will it take to validate my Web site?

Validation processing time depends on multiple factors like the number of your Web pages, the complexity of your page content, your server's capacity, and your network bandwidth. As an example, an average Web site of 900 pages could be processed in 15 to 20 minutes.

Can I check internal websites or sites that require authentication?

No. Sites must be publicly accessible to be validated.

My site uses Javascript. Is that OK?

Some users do not enable Javascript in their browsers. Therefore, to be fully interoperable your Web site should not depend on navigation links inside the Javascript. W3C Validator Suite® will not process the Javascript; it will only follow the links that are specified in the HTML markup. If the Javascript modifies the HTML, W3C Validator Suite® will only check the original, unmodifed HTML. Aside from this, Javascript in your site should not impact the W3C Validator Suite® report.

Is the W3C Validator Suite® published under an open-source license?

No.

Is there an API?

Currently the W3C Validator Suite® is only available as Software-as-a-Service from https://validator-suite.w3.org. We are hoping to offer an API in the future which will make it possible to integrate with Content Management Systems (CMS) or an existing Quality assurance (QA) process.

How can I submit bug reports or send ideas?

W3C Validator Suite current customers should login and follow the "Support" link available from the top menu. Other comments and suggestions should be submitted to by email to our support team.
