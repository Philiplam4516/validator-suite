Frequently Asked Questions

  1. When I see the W3C Validator Suite brand logo, what does it mean?
  2. When I see the W3C Validator Suite badge, what does it mean?
  3. Do the brand logo/badge imply validity?
  4. Must I include the badge on every validated page?
  5. May I use the badge on pages that have not be run through the W3C Validator Suite?
  6. May I use the brand logo off the Web for creative purposes (e.g., t-shirts, stickers, etc.)?
  7. May I use the brand logo/badge on commercial products (e.g., my own validator service)?
  8. May I create derivative works of the brand logo/badge?
  9. My site uses Javascript. Is that OK?

1. When I see the W3C Validator Suite brand logo, what does it mean?

The brand logo -- 5 check marks of success for validity, accessibility, compatibility, optimization, efficiency -- refers in a broad sense to the W3C Validator Suite and may be used to promote it. There are no implications of usage or validity.

2. When I see the W3C Validator Suite badge, what does it mean?

The badge indicates that a page (or site) has been evaluated with the W3C Validator Suite. The badge does not imply anything about the outcome of running the W3C Validator Suite.

3. Do the brand logo/badge imply validity?

No. The logo/badge do not imply anything about the outcome of running the W3C Validator Suite.

4. Must I include the badge on every validated page?

No, though you may do so if you wish. You may include the logo on one page and list or describe pages that have been evaluated through the W3C Validator Suite.

5. May I use the badge on pages that have not be run through the W3C Validator Suite?

No. You may use the brand logo if you wish to refer to the W3C service.

6. May I use the brand logo off the Web for creative purposes (e.g., t-shirts, stickers, etc.)?

Yes. We are interested in creative uses of the logo that show support for the service. Please let us know about your usage of the brand logo off the Web at validator-feedback@w3.org.

7. May I use the brand logo/badge on commercial products (e.g., my own validator service)?

Not at this time.

8. May I create derivative works of the brand logo/badge?

Not at this time.

9. My site uses Javascript. Is that OK?

Some users do not enable Javascript in their browsers. Therefore, to be fully interoperable your Web site should not depend on navigation links inside the Javascript. W3C Validator Suite will not process the Javascript; it will only follow the links that are specified in the HTML markup. If the Javascript modifies the HTML, W3C Validator Suite will only check the original, unmodifed HTML. Aside from this, Javascript in your site should not impact the W3C Validator Suite report.
